A computer System.

A computer system is a key tool in programming and it may be defined as an electronic device which stores and processes data.

Data fed into the computer system is input data and data generated by the system is output data.

A computer system is made up of two main components:

a) Computer hardware
- Referring to the physical components of the computer system.

b) Computer Software
- Referring to the programs which run within the computer System.

The hardware and the software components of the computer System are equally important in enabling a computer perform a specific task.

Examples of computer hardware and software.

  • keyboard
  • speakers
  • monitor
